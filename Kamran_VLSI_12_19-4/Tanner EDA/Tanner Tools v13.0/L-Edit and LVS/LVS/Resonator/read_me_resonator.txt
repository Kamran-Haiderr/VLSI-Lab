Element Description File Example:

This file documents a example of using an element description file.  Refer to the LVS manual for a
more thorough description of this example.

Command Line Invocation:
lvs layout_resonator.spc schematic_resonator.spc -o resonator.lvs -l resonator.lst -e resonator.elm -nrcl -vfa -faf

GUI Settings:
Layout netlist: layout_resonator.spc
Schematic netlist: schematic_resonator.spc
Output file: resonator.lvs
Node and element list: resonator.lst
Display Options - Output File
  * Show fragmented classes
  * Show automorph classes
Device Terminals
  * Consider M bulk terminals and B, J, Q, Z substrate terminals

No errors are reported but a warning is given to notify the user that the circuits are checked only 
as far as the element file dictates.
